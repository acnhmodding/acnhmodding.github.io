A open-source tool created by [@KillzXGaming](https://github.com/KillzXGaming) to edit _many_ file formats used by Nintendo consoles.  
> GitHub: [KillzXGaming/Switch-Toolbox](https://github.com/KillzXGaming/Switch-Toolbox)

## BFRES Support
- Full editing support for Wii U and Switch formats
- Import models from DAE, FBX, OBJ, CSV
- Edit rendering/material settings and texture bindings
- Copy materials between models/files
- Export/import animation and model sections
- Manage FMDL objects (sub-meshes)
- Preview skeleton and material animations (SRT0, FSKA, FMAA, etc.)
- Experimental GIF-to-texture animation converter

## Setup

<Steps>
    1. Download the latest version from [GitHub Releases](https://github.com/KillzXGaming/Switch-Toolbox/releases)
    2. Open a supported file (e.g. `.bfres`, `.bntx`, `.bars`, etc)  
    3. Use the toolbox to view or edit models, textures, animations, audio, and materials.
    4. Save your edited files and test in-game!
</Steps>

> For advanced tutorials (e.g. editing KCL, AAMP, or BFRES), refer to the [Wiki](https://github.com/KillzXGaming/Switch-Toolbox/wiki)
